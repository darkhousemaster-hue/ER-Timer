ER Timer - Handbook

Everyday guide for running games. (Manager settings are password-protected and not covered here.)

1Windows & Always-on-Top

ER Timer runs as two kinds of window.

  • Control window - the panel you operate (timer and hints).
  • Player window - the large countdown your players see, one per room.

The Control window is always-on-top: it stays visible above other programs, so clicking another app (a browser, music player, etc.) will never hide or bury it.

Closing the Control window shuts down the whole app, including the player screen(s).

2Minimize to Icon Button FEATURE

Tuck the panel away without losing it - and without it dropping behind other windows.

  1. Click the window's minimize button. The Control window disappears.
  2. A small floating icon button (the size of a desktop shortcut, showing the ER Timer icon) appears in the bottom-right corner of your main screen.
  3. That button always stays on top of every other program, so you can always find it.
  4. Click the icon button and the Control window reopens at the exact same size and position it had before.
Good to know: while minimized, the timer keeps running and players still see their countdown - only your control panel is hidden. The button has no taskbar entry, so it won't clutter your taskbar.

3The Timer

A four-digit MM:SS countdown, mirrored on the player screen.

Setting the time

Each digit has a + above it and a - below it. Tap them to dial in minutes and seconds before you start.

Running it

ButtonWhat it does
StartBegins the countdown from the set time.
PauseFreezes the countdown; press Start to resume.
ResetStops and returns to the default start time.
The countdown runs in the app's core process, so it keeps perfect time even when minimized or when other windows are busy.

4Rooms & Fullscreen

Run one room, or two side by side.

  • If a second room is enabled, switch between Room 1 and Room 2 tabs. The hints below always apply to the room tab you have selected.
  • The ⛶ (fullscreen) button toggles the player screen for the current room between windowed and full screen.

Turning the second room on or off is done in password-protected Manager settings.

5Text Hints

Send a written clue to the player screen.

  1. Type your clue in the "Type a hint..." box.
  2. Click Send to show it on the player screen for that room.

Hint sound

  • The Hint sound checkbox controls whether a sound plays on the player screen each time you send a hint.
  • ▶ Test plays that sound once so you can check it.

6Picture Hints

Push a prepared image to the player screen with a single click.

Picture hints are organised into Phases, and each phase holds Riddles, and each riddle holds images. (This library is set up in advance in Manager Mode.)

  1. Click a Phase to expand it, then click a Riddle to reveal its images.
  2. Click the image you want - it appears on the player screen immediately, and the chosen thumbnail is highlighted.
  3. Click a different image to switch; the new one replaces the old.
Picture hints are limited to still images (PNG, JPG, GIF, WEBP, BMP). Video hints are not supported.

7Clearing Hints

ButtonWhat it does
Remove pictureTakes the current image hint off the player screen (text is left alone).
Remove allClears both the text hint and the image hint for that room at once.

8Manager Mode

The 🔒 Manager Mode button opens the password-protected settings area.

Everything behind that password - building the picture-hint library, backgrounds, sounds, colours, number styles, displays, default reset time and more - is intentionally left out of this handbook, which covers only what you use during a normal game.

9Tips

  • Set your countdown before the game starts, then just press Start when players enter.
  • Expand the right Phase/Riddle ahead of time so the picture you need is one click away.
  • Minimize to the icon button between games to free your screen; the timer keeps running.
  • The Control window stays on top - you never have to hunt for it behind other apps.
